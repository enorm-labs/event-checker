# Vision / Roadmap / Ideas

* Like Resident Advisor, Bandsintown, eventbrite, and songkick for Berlin events, but better
* Focus on all (and small) music events (concerts, club nights, festivals etc. - not only Techno/Electronic music) in venues (clubs, concert halls etc.) of
  Berlin (maybe
  later also other cities)
* Form for "Venue or event missing? Let me/us know!" (Link to GitHub issues?)
* Feedback Form? (Or link to GitHub issues)
* Tickets -> GitHub issues
* Takeover more ideas from here: https://docs.google.com/document/d/1UkxdJECxvB6noW-n8dzX-r0du18M9-ggp6iWeapHpvI/edit?tab=t.0#heading=h.32gy0wklnx78
* Later:
    * Login / Profile
        * Favorite Venues
        * Favorite Artists
        * Notifications
        * User Profile for customizing (show only favorite venues, order venues, customize start page etc.)
        * User can say that he is interested or going to an event (RSVP)
        * Recommendations?
        * Allow users / venues to add events? (Review before publishing) --> User Management and RBAC (keycloack)
    * Social Network?
        * Connect with Friends
        * See which events your friends are interested in or going to
    * Venue profile (with links)
    * Artist profile (with links)
    * Rank events by popularity (amount of RSVPs)
    * Rank events by artist popularity
    * Integrate Spotify, Deezer, Soundcloud, Resident Advisor for getting notifications when favorite artists perform at a venue
    * Create Club Map (Events nearby)
    * API for third party apps --> API Management (API subscriptions, API keys etc.)
